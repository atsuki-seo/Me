## Gitの基本コマンド

```bash
# リポジトリの初期化
git init

# 変更のステージング
git add .

# コミット
git commit -m "メッセージ"

# プッシュ
git push origin main
```

### ブランチ操作

- `git branch feature`：ブランチ作成
- `git checkout feature`：ブランチ切り替え
- `git merge feature`：マージ
