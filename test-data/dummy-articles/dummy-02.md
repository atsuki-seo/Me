## Swiftとは

SwiftはAppleが開発したプログラミング言語で、iOS/macOSアプリ開発に使用されます。

### 基本文法

```swift
// 変数と定数
var mutableValue = "変更可能"
let constantValue = "定数"

// 関数定義
func greet(name: String) -> String {
    return "Hello, \(name)!"
}
```
