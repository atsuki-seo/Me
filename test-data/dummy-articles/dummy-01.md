## プログラミングとは

プログラミングとは、コンピュータに対して命令を与えることで、特定の処理を自動化する技術です。

### 基本的な概念

- 変数：データを一時的に保存する箱
- 関数：処理をまとめたもの
- 条件分岐：条件によって処理を分ける

```python
# サンプルコード
def hello(name):
    return f"Hello, {name}!"
```
